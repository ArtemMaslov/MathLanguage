1. Главная функция
	Синтаксис: 
		<Ключевое слово> [%s].
			[Условие теоремы]
		<Начало тела функции>:
			<Код>
		<Конец тела функции>.
	Ключевое слово:
		1) Задача.
		2) Теорема.
	Условие теоремы:
		Ничего не значащая строка. Может быть опущена.
		Примеры:
			1) Если a = 10 и b = 20, то выражение 2 * a + b * 5 равно 120.
	Начало тела функции:
		1) Доказательство.
		2) Приведём доказательство этого утверждения.
		3) Докажем это утверждение.
		4) Начнём доказательство.
		5) Решение.
	Конец тела функции:
		1) Что и требовалось доказать.
		2) Ч.т.д. / ч.т.д.
		3) Задача решена.
	Примеры:
		1) Теорема.
		2) Теорема 1.2.3.4.
		3) Теорема о непрерывной функции.
		4) Теорема о непрерывной функции № 1.4.5.3.

4. Объявение переменных
	Синтаксис:
		<Ключевое слово> <Имя переменной> <Присвоение> <Вычислимое выражение>.
	Ключевое слово:
		1) Существует
		2) Пусть
		3) Найдётся
	Имя переменной:
		Начинается с латинской буквы (прописная/строчная).
		В коде должно быть с двух сторон выделено пробелами.
		Содержит:
		1) A-Z, a-z, А-Я, а-я.
		2) Символ звёздочка *
		3) Символ штриха '
		4) _
		5) +, -
	Присвоение:
		1) =
		2) равняется / равно / равное / приравняем/ ...
		3) 
	Примеры:
		1) Пусть a = 10.
		2) Существует b равное 20 * a.
		3) Очевидно, что найдётся c, такое что c = 2 * a + b * 5.
		4) x1 = (-b + sqrt(D))/(2*a)
		

5. Условный оператор
	Синтаксис:

			<Код>
			

6. return
	Примеры:
	1) Закончить решение задачи.
	2) Закончить доказательство.
		
6. Стандартные функции
	1) Вывод в консоль
		Синтаксис:
			<Ключевое слово> <Имя переменной>.
		Ключевое слово:
			1) Вычислим
			2) Получим значение
		Примеры:
			1) Вычислим значение c.
			2) Ответом является вычислимое выражение -c/b.
	
	2) Ввод чисел
		Синтаксис:
			<Ключевое слово> <имя переменных>.
		Ключевое слово:
			1) Возьмём произвольные.
			
	3) Вывод строк
		Синтаксис:
			"Строка".
		Примеры:
			1) Задача имеет "беск. число решений".
			2) "Решений нет".
